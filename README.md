# Customer Buying Trend Prediction (ML)

This project predicts whether a customer will buy a product based on features like age, income, visits, and past purchases using a Machine Learning model.

## Tech Stack
Python, Pandas, Scikit-learn, MLPClassifier

## Workflow
- Load dataset (CSV)
- Split features & target
- Train-test split
- Feature scaling
- Train ML model
- Predict customer buying behavior

## Output
Predicts: Buy (1) or Not Buy (0)
